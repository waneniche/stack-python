## Penjelasan Program

Program ini merupakan implementasi struktur data **Stack (LIFO)** 
menggunakan `collections.deque` dari Python.

Alasan menggunakan `deque`:
- Mendukung operasi push dan pop dengan kompleksitas O(1)
- Lebih efisien dibanding list untuk operasi di ujung data
- Merupakan library bawaan Python

Program dijalankan melalui terminal dan menyediakan menu interaktif
untuk melakukan operasi stack seperti push, pop, dan clear.
